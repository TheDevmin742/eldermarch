=========================================================

  2. Install Claude Code: follow https://code.claude.com  (one command, needs Node)
  3. Open a terminal IN THIS "Website" FOLDER and run:  claude

  5. It ends by giving you a live URL like
     https://YOURNAME.github.io/eldermarch/  → pin it in the club Discord.
